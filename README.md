NayePankh Foundation Website Redesign

Overview

This project is a responsive website redesign created for the NayePankh Foundation Web Development Internship Selection Process.

The website highlights the foundation's mission, programs, internships, volunteer opportunities, testimonials, gallery, and contact information.

Features

- Responsive Design (Desktop, Tablet, Mobile)
- Modern Hero Section
- About Us Section
- Programs Section
- Internship Opportunities
- Volunteer Registration Form
- Testimonials Section
- Gallery Section
- Contact Information
- Dark Mode Toggle
- Mobile Navigation Menu
- Smooth Animations

Technologies Used

- HTML5
- CSS3
- JavaScript

Purpose

This project was developed as part of the NayePankh Foundation internship selection process to demonstrate front-end web development skills.

Note

Images and logo belong to NayePankh Foundation and are used for educational and internship demonstration purposes only.
